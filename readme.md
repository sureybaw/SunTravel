# Proyecto Web - Agencia de Viajes SunTravel

Este proyecto consiste en el desarrollo de un sitio web para una agencia de viajes ficticia llamada **SunTravel**, realizado utilizando **HTML5 y CSS3**.

El sitio web incluye diferentes secciones con destinos turísticos organizados por continentes, así como un formulario de solicitud de presupuesto y una página de contacto.

## Estructura del proyecto

El proyecto está organizado en los siguientes archivos:

- index.html → Página principal
- america.html → Destinos en América
- europa.html → Destinos en Europa
- africa.html → Destinos en África
- asia.html → Destinos en Asia
- presupuesto.html → Formulario de solicitud de presupuesto
- contacto.html → Información de contacto
- css/style.css → Hoja de estilos del sitio web
- img/ → Carpeta que contiene las imágenes del sitio
